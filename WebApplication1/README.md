&nbsp;							**Recipe** **Writer**





* Lancer votre base de données.
* Ouvrez la solution dans Visual Studio.
* Dans le fichier appsettings.json:

&nbsp;	- Ajoutez l'adresse, identifiant et mot de passe de votre base de données.

Lancez le compilateur.



La première page affiche les recettes existantes en base de données.



Cliquer sur Accueil affichera le logo.



Cliquer sur le petit logo en haut à gauche vous ramènera à cette page.



Cliquer sur Ajouter recette vous permet de créer une recette, en y indiquant, les détails de celle-ci.



Cliquer sur liste des ingrédients vous permets de visualiser les ingrédient déjà présent en base de donnée.

D'en ajouter, et d'en supprimer.
